📌 Project Title

Django Portfolio Website

🚀 Description

This is a fully responsive personal portfolio website built using Django framework. The project showcases my skills, projects, and contact information.

The website includes authentication system, dynamic project management, and database integration using MySQL.

🛠 Tech Stack

• Backend: Python, Django

• Frontend: HTML, CSS, JavaScript

• Database: MySQL

• Version Control: Git, GitHub

✨ Features

• User authentication (Login / Logout)

• Admin panel for managing content

• Dynamic project display

• Contact form

• Responsive design

• Database integration

⚙ Installation

‣ git clone https://github.com/yourusername/django-portfolio-website.git

‣ cd django-portfolio-website

‣ pip install -r requirements.txt

‣ python manage.py migrate

‣ python manage.py runserver
